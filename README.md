## Task: `functions.sh`
**Description:**  
first function: Develop a pow() function that takes two arguments (a, b) and raises the first argument to the power of the second (a^b).
Second function: Develop the shortest() function, which can take an unlimited number of arguments(strings) and output the shortest argument.
Third function: Develop a print_log() function that takes a string as an argument and outputs the same string with the date at the beginning. In order for the automatic check to work, the string must be in this format: YEAR-MONTH-DAY HOUR:MINUTES.
